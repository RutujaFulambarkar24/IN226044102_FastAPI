# 💊 Medical Appointment System (FastAPI)

## 📌 Overview

A backend system built using FastAPI to manage doctors and patient appointments.
The application supports appointment booking, workflow management, and advanced data operations like search, sorting, and pagination.

---

## 🚀 Features

* Manage doctors (create, update, delete with constraints)
* Book appointments with validation and dynamic fee calculation
* Appointment lifecycle management (confirm, cancel, complete)
* Filter doctors based on specialization, fee, experience, and availability
* Search functionality across doctors and appointments
* Sorting and pagination for efficient data handling
* Combined browsing endpoint (search + sort + pagination)

---

## 🧠 Core Concepts

* RESTful API design
* Pydantic validation
* Query parameters & filtering
* CRUD operations
* Business logic implementation
* Workflow/state management
* Pagination and sorting
